hexo 블로그 만들기 1탄. 개발환경 구축하기

hexo는 사용하기 간편한 프레임워크로 다양한 운영체제에서 사용가능하다.
내가 선택한 운영체제는 Linux이며, 버전은 ubuntu 64-bit이다.

Part1. Linux 환경 구축

가상 머신 설치하기

linux 운영체제를 설치하기 위해선 가상 머신이 필요하다.
대표적인 가상머신으로는 vmware, virtualbox 등이 있는데, 나는 virtualbox만을 써봤다.
아직까진 아무런 불편한 없이 매우매우 잘 사용하고 있다.

  1. virtualbox download
  2. Windows hosts->VirtualBox-xxx-Win.exe 파일 다운로드
  3. execute 파일 실행 및 설치

우분투 다운로드

가상 머신 설치를 완료했다면, linux-ubuntu 파일을 다운받는다.
linux version 중 ubuntu로 개발한다는 가정하에
utuntu 16.04.3 LTS download

linux 시스템 구축

  1. virtualbox 실행
  2. 새로 만들기 클릭
  3. 자신의 컴퓨터 버전과 같은 linux version 선택. ex) Linux Ubuntu 64-bit

그러나, 리눅스 환경이 처음인 분들은 git, npm, nodejs 환경도 함께 구축해야한다.
